### 👊说明

本项目是基于Vue 3 + Typescript + Vite2.0+axios+Vant3+Vuex构建一个基础的入门项目，

###### 已经配置好相关插件，无需再重复下载，开箱即用

## 技术点

- 版本：vue3.0(Ts)
- 路由：vuex
- 构建工具：vite
- 网络请求：axios
- 移动端UI框架：vant(Ts)
- 移动端适配解决方案：postcss-pxtorem

## 项目运行

```
# 克隆项目到本地
github.com/Tianyuwen/my-vue-app.git

# 进入文件夹
cd my-vue-app

# 安装依赖
npm install或 npm i(简写)

# 开启服务
npm run dev

# 打包项目
npm run build

```
